# Livestream Coffee — Digital Menu

Livestream Coffee Menu — A sleek, mobile-friendly digital menu for Livestream Coffee, Vesu. Built with HTML/CSS/JS featuring 3D aesthetics, smooth scroll navigation, brand-accurate maroon & gold theme, and all 73 menu items across food and specialty coffee categories.

## Tech Used
- HTML5, CSS3, Vanilla JS
- Intersection Observer API
- Google Fonts (Barlow Condensed)
- Fully Mobile Responsive
